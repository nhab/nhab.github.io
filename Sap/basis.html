Basis is a set of programs and tools that act as an interface with
Database, Operating system, Communication protocols and business applications (such as FI, CO, MM,etc).
Full form of BASIS is "Business Application Software Integrated solution"
SAP applications such as FI,CO,PP etc. can run and communicate with each other


across different Operating systems and Databases with the help of BASIS.
Nowadays Basis is known as Netweaver.
Alias of BASIS is SAP Application Server Technology and alias of NetWeaver is SAP Web Application Server.

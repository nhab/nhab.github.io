Disable chrome annonying notifications : three-dot menu button > Settings > Advanced > Content Settings > Notifications.
fixing chrome font rendering problem : settings ( chrome://settings/ in addressbar) > advanced > system > "Use hardware acceleration when available" to off > Restart Google Chrome.
Chrome slow image rendering : chrome://flags in the address bar > GPU rasterization (or load chrome://flags/#enable-gpu-rasterization ) > Switch to enabled > Restart Google Chrome..

Disable chrome annonying notifications :  three-dot menu button > Settings > Advanced > Content Settings > Notifications.

fixing chrome font rendering problem :   settings ( chrome://settings/ in addressbar) > advanced > system > "Use hardware acceleration when available" to off >  Restart Google Chrome.

 also  chrome://flags > 
              Override software rendering list - Enable
              Enable display list 2D canvas - Enabled
              Enable fast tab/window close - Enable


   also in windows :  

      Step 1: Use the text tuner         : Windows > Start menu > type ClearType > Adjust ClearType Text >check the box "Turn on ClearType." >Next to complete the steps.

      Step 2: Windows appearance settings: Windows > Start menu > type Appearance> Adjust the appearance and performance of Windows> uncheck "Smooth edges of screen fonts" > Apply> restart chrome.



Chrome slow image rendering :   chrome://flags in the address bar > GPU rasterization (or  load chrome://flags/#enable-gpu-rasterization ) > Switch to enabled > Restart Google Chrome..

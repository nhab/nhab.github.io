Environment Setup

1-Install Node.js
2-Create the Root Folder 

    C:\\Users\\username\\Desktop>mkdir reactApp
    C:\\Users\\username\\Desktop\\reactApp>npm init
3-Install Global Packages 

  C:\\Users\\username\\Desktop\\reactApp>npm install -g babel
  C:\\Users\\username\\Desktop\\reactApp>npm install -g babel-cli
4 - Add Dependencies and Plugins 

  C:\\Users\\username\\Desktop\\reactApp>npm install webpack --save
  C:\\Users\\username\\Desktop\\reactApp>npm install webpack-dev-server --save
The --save command will add these packages to package.json file. 

  C:\Users\username\Desktop\reactApp>npm install react --save
  C:\Users\username\Desktop\reactApp>npm install react-dom --save
